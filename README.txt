/* Greenify */

Greenify is a multi-step web application that calculates users' estimated annual carbon footprint based on their daily lifestyle habits.

* Features:

 -> Multi-Step Calculation
 -> Detailed Results
 -> Visual Analysis
 -> Personalized Recommendations
 -> Scenario Analysis
 -> Mobile-Friendly Design

* Technologies Used:

 -> HTML/CSS (Tailwind CSS)
 -> JavaScript
 -> Chart.js

* Developer: Eren Söğütlü

LINK: https://erensogutlu-greenify.netlify.app/
